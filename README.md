# Git Fundamentals Lab

Projeto desenvolvido para estudar, praticar e documentar os principais fundamentos do Git e GitHub de forma prática e organizada.

Este repositório funciona como um laboratório de versionamento, contendo exemplos reais, desafios, anotações e testes de workflow utilizados no desenvolvimento profissional de software.

---

# Objetivos

- Aprender os fundamentos do Git
- Entender o funcionamento do versionamento
- Praticar comandos essenciais e avançados
- Simular workflows profissionais
- Melhorar organização de projetos
- Aprender boas práticas de commits
- Desenvolver experiência com branches e merges
- Documentar estudos de forma profissional

---

# Tecnologias e Ferramentas

- Git
- GitHub
- Markdown
- PowerShell
- Linux Terminal
